# 校正設計工具 calib_designer

設計與驗證**校正（raw → target，原始讀值 → 目標真值）**的桌面工具。輸入一組量測點
（x 原始、y 目標），並排比較兩種校正方式的誤差，挑出最適合塞進韌體的那一種。

兩種校正方式：

- **分段線性查表 (LUT)**：用 N 個斷點的折線內插。適合非線性、要逐段貼合的資料。
- **線性回歸**：單一直線 `y = a·x + b`（gain/offset）。適合接近線性、只想存兩個常數的資料。

典型用途：電流/電壓量測校正表、感測器線性化、溫度補償等。

## 啟動

```powershell
python main.py            # 開發用，stderr 直接看得到
.\calib_designer.bat      # 雙擊：優先 pythonw（無 console）
```

也可從 jack-toolkit launcher 啟動（本目錄含 `manifest.json`，顯示名稱「校正設計工具」）。

> 依賴 **matplotlib**（會一併帶入 numpy）。請先跑專案根的 `install_requirements.bat`，
> 或 `python -m pip install matplotlib`。缺套件時啟動會跳 messagebox 提示。

## 校正方式

左側「校正方式」切換 **分段線性查表 (LUT)** 或 **線性回歸**（= 主方法，決定匯出與圖中主線）；
旁邊「公式」按鈕列出兩者的標準數學式。

**圖表比較項目**：主方法一律顯示，另可勾選同時疊上 **線性內插 / 線性回歸 / 均勻 / 原始(校正前)**
任意組合，上下圖（疊圖、誤差）都會跟著畫。主方法那一項固定勾選。

## LUT 節點選法

**自動**：兩種模式 × 兩種演算法。

| 模式 | 你給 | 工具回 |
|---|---|---|
| 給點數，找最佳位置 | 節點數 N | N 個節點的最佳擺放 |
| 給目標最大誤差，找最少點數 | 容許誤差 E（量度為相對時即「% 以內」） | 達標所需的最少節點 |

- **演算法 1（逐步插點 greedy）**：每次把節點插到誤差最大處，快、近似最佳。
- **演算法 2（全域最佳 DP minimax）**：給定點數下最大誤差最小的證明解；O(n³)，>600 點會拒絕。
- 「說明」按鈕詳述兩者差異（演算法 2 的最大誤差一定 ≤ 演算法 1，但 RMS 不一定）。

**手動插點模式**（勾選後）：在 datagrid 的「節點」欄點選自己的斷點，與演算法並排比較。
點數上限 = 演算法點數（超過會擋下並 beep），確保公平比較；至少留 2 點。**頭尾節點可自由
取消**——落在節點範圍外的點，評估與匯出的 C 都比照韌體外插（低於表頭由原點 (0,0)、高於
表尾沿最後一段、負值箝位為 0），所以工具報的誤差跟現場一致。

## 線性回歸：擬合點選擇

線性回歸模式下會多出「回歸取樣點數」滑桿，且 datagrid「節點」欄變成「要餵進最小平方擬合
的點」：

- **預設點數 = LUT 的節點數，採均勻撒點**。
- 拉「回歸取樣點數」滑桿改變點數（均勻重撒）。
- 也可在「節點」欄手動點選個別點（**不受 LUT 點數限制**，至少留 2 點）。

無論用幾點擬合，誤差一律對**全部樣本**評估，這樣才看得出該直線在每一點的表現。

## 兩種誤差量度

- **絕對誤差** `|Δy|`：規格寫成 ±X 單位時用。
- **相對誤差 %讀值**：量測值跨多個級距（如電流 mA→A）時用，避免小值端被大值端壓著跑。

## 逐點比較數據表

「結果摘要」下的 **逐點比較數據表** 按鈕，列數 = 資料點數，三方並排：

| 欄 | 意義 |
|---|---|
| 原始(MCU)、目標 | x、y |
| 原始誤差 | 校正前 `y - x` |
| 內插計算值、內插誤差 | 線性內插（LUT）的計算值與帶號 `目標-計算值` |
| 回歸計算值、回歸誤差 | 線性回歸同上 |
| 最佳 | 該點 \|誤差\| 最小者；**只有這一欄上色**（紅=原始、藍=內插、綠=回歸） |

頂部摘要列出 原始 / 內插 / 回歸 三者的 max、RMS 與各自「最佳」點數。可「匯出 CSV」。

## 匯出

- **C 程式碼**：
  - LUT → 單組查表（`cal_<組名>_OV_X/TV_Y` + `linearInterpolationFromLUT`），查表邏輯完全
    對齊現場韌體：表內線性掃描、低於表頭由原點外插、高於表尾沿末段外插、負值箝位為 0。
  - 線性回歸 → `CAL_<組名>_GAIN/OFFSET` + `linearCalFromReg`（亦負值箝位為 0）。
  - 「C 組名」欄可填（如 `12V_curr`）；彈窗可複製或存檔。
- **CSV 節點表**：主方法 LUT 的節點 x、y、每段最大誤差。
- **PNG 圖**：把目前疊圖與誤差圖存成圖檔。

右側圖表附 matplotlib 工具列，可縮放 / 平移 / 另存。
`engine.firmware_interp_single()` 是韌體查表邏輯的 Python 鏡像，可驗證任意輸入與 MCU 端一致。

## 圖示

`make_icon.py` 一次產生 `calib_designer.ico`（視窗，多尺寸）與 `icon.png`（48x48，launcher
卡片）。需要 Pillow（matplotlib 會一併帶入）；runtime 載入圖示不需要 Pillow。

## 檔案

| 檔 | 內容 |
|---|---|
| `main.py` | 進入點 + 全域 excepthook（錯誤可見性） |
| `app.py` | 主視窗 UI、matplotlib 嵌入、各方法比較與匯出 |
| `engine.py` | 演算法核心（貪婪 / DP / 均勻 / 線性回歸 / 評估 / 比較 / 匯出產生器），純 Python 可單測 |
| `theme.py` | 配色與小工具工廠 |
| `電流校正範例.csv` | 範例資料（x=原始, y=目標） |
